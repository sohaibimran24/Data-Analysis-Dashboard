#  Data Analysis Dashboard

A comprehensive financial data analysis dashboard built with Python.

##  About The Project
Analyzes financial data including cryptocurrency prices, stock market trends, personal expenses, and market correlations. Generates insights, visualizations, and statistical reports.

##  Features
- Load and analyze CSV data
- Statistical analysis (mean, median, std dev, etc.)
- Price trend analysis
- Portfolio performance tracking
- Correlation analysis
- Data visualization with charts and graphs
- Risk assessment metrics

##  Built With
- Python 3
- Pandas (data manipulation)
- NumPy (numerical computing)
- Matplotlib (data visualization)

##  Installation
pip install pandas numpy matplotlib
python dashboard.py
